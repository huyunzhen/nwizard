# nwizard
http://github.com/ricemouse/nwizard.git

## Installation
git clone http://github.com/ricemouse/nwizard.git

## To compile

```
cd nwizard
npm install
cd app/spi
bower install
gulp (optional, used to build executable binary files, but still not reliable)
```
## To run
install nwjs latest version
and then run
```
nw ./app
```
