Updates Location

You can ensure that you have the latest updates by checking the web site. You will be able to select which updates you want before they are downloaded. Once downloaded, updates will be stored in the local directory indicated below.

All updates that are newer than the currently installed version will be installed. This default will only update already installed drivers and firmware, and will neither resolve unknown devices, nor update drivers that are not running.
Also select updates that are not installed or undetected
Select this option in order to attempt to detect all possible hardware in the system. This should be selected only if you have some unknown devices in Windows and that you want to resolve.
