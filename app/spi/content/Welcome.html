Welcome to System Pack Installer

This wizard allows you work with system updates for your machine. You can apply updates to the local system now, download updates to be applied later, or download updates for other systems to be applied remotely (CLI only).
The updates can be either System Packs or the latest individual updates. System packs contain a collection of updates that have been tested for compatibility with one another, and are the preferred delivery for updates.

To begin the update process, click Next.
